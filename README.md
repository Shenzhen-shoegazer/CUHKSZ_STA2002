# STA2002: Probability and Statistics II 概率与统计 II

## 简介 (Introduction)

这个仓库包含了香港中文大学（深圳）STA2002《概率与统计 II》课程的课后学习笔记。所有笔记均以 Jupyter Notebook 格式提供，方便运行代码与查看数学公式。

This repository contains the study notes for the STA2002 'Probability and Statistics II' course at CUHK-SZ. All notes are provided in Jupyter Notebook format for easy code execution and mathematical formula rendering.

## 使用方法 (Usage)

你可以直接在 GitHub 上查看这些 `.ipynb` 文件。如果需要运行其中的代码，请确保你的本地环境安装了 Python 以及 Jupyter Notebook 或 JupyterLab。

You can view these `.ipynb` files directly on GitHub. To run the code, ensure you have Python and Jupyter Notebook or JupyterLab installed in your local environment.

```bash
# 安装 Jupyter (Install Jupyter)
pip install notebook

# 启动 Jupyter Notebook (Start Jupyter Notebook)
jupyter notebook
```
